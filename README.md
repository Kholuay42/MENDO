# Mendo Service Management

Dies ist ein Webprojekt, das ich als Freelancer für einen Kunden umgesetzt habe.

## 🛠 Verwendete Tools & Technologien
* **HTML5 & CSS3**: Struktur und Design der Webseite.
* **JavaScript**: Für interaktive Elemente.
* **VS Code (Live Server)**: Lokale Entwicklungsumgebung.
* **FileZilla**: FTP-Deployment.
* **Hosting**: **STRATO** (Einrichtung und Verwaltung des Webspaces).

## 🚀 Besonderheiten & Workflow
Im Fokus stand ein sauberes UI/UX Design, das die Dienstleistungen des Kunden klar strukturiert darstellt. 
Der Workflow umfasste die lokale Entwicklung in VS Code, das Testen über den Live Server und das anschließende Deployment via FileZilla auf den STRATO-Server.

## 🔗 Live-Vorschau
Die fertige Webseite ist unter [mendoservice.de](https://mendoservice.de/) erreichbar.
